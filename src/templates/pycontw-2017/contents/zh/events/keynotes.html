Andrew-Godwin (andrewgodwin)

@andrewgodwin
http://www.aeracode.org/

Bio

Andrew 是一位 Python 程式設計師、Django 核心開發人員、同時亦是 South 與 Django migrations 框架作者。在程式設計、維護與機器集中管理擁有十多年經驗。

他是牛津大學莫德林學院 (Magdalen College, Oxford) 的出色校友，現居風景怡人的舊金山。目前則任職於 Eventbrite。

Andrew 是一位相當喜愛旅遊的演講者，興趣是偶爾騎騎單車、射箭、飛行小型飛機、爬山和奶酪。

{# Carol-Willing #}

Carol Willing (willingc)

@WillingCarol
https://willingconsulting.com/

簡歷

經常總會聽到 Carol Naslund Willing 說：“這真是太酷了。 它是如何運作的？好希望學習到更多有關的事物”。她熱愛開發軟體，目前為 PSF (Python Software Foundation) 基金會理事之一 、 Project Jupyter 的核心開發人員與加州理工大學聖路易斯奧比斯 ( Cal Poly San Luis Obispo) 的軟體研究工程師。她活躍於開源專案的共同開發，並且維護像是 OpenHatch 與 Anita Brog Institute 的開源專案。

與人之間的經驗使我有了這樣感觸：“用你所學，使事情更好，並且與人分享”。Caril Wiilling 這樣說的。

Carol Willing 對學習和分享充滿熱情。最近的演講包括 PyCon US 2016，PyData Carolinas 2016，PyCon Philippines 2016。

{# Russell-Keith-Magee #}

Russell Keith-Magee

@freakboy3742 http://cecinestpasun.com/

簡歷

Russel Keith-Magee 博士是擁有 10 年經驗的 Django 核心團隊老手，且有五年時間一直擔任 Django 軟體基金會 (Django Software Fundation) 的主席，他是 BeeWare 專案的發起人，透過這個專案 GUI 工具用來支援 Python 軟體開發。同時也是 Django Evolution 的共同發起人。

他畢業於澳洲科廷科技大學 (Curtin University of Technology)，取得物理學士學位且獲得計算機科學的一級獎章 (First Class Honours)。之後 Russell 完成了關於學習模型調查與類神經網路的博士論文研究並且取得哲學博士學位。

自稱為 geek，喜歡新得想法和推廣可能使用的技術與科技，因此他認為資訊科技是充實現實與物理、概念和金融，而他也熱衷在建構這方面技術。
